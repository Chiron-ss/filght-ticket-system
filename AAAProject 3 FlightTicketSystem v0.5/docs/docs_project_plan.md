# 项目计划: 航班票务管理系统

## 项目描述
这是一个基于 Qt 和数据库开发的航班票务管理系统。系统功能包括用户注册、查票、订票以及管理订单，同时包含客户端和服务器端。

## 模块计划
### 客户端功能模块
1. 用户登录与注册界面。
2. 航班查询界面。
3. 订单管理界面。

### 服务器端功能模块
1. 处理用户请求（登录、注册、查询）。
2. 管理航班信息。
3. 处理订单创建和状态更新。

### 数据库设计
- `Users` 表: 存储用户信息。
- `Flights` 表: 存储航班信息。
- `Orders` 表: 存储用户的预订信息。

## 开发阶段
- 第一阶段: 启动项目，完成环境搭建。
- 第二阶段: 搭建客户端和服务器的通信逻辑。
- 第三阶段: 完善功能模块，进行集成测试。
